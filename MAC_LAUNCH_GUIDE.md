### 🛠️ 故障排除

#### 如果双击无效
1. 确保文件权限正确：
   ```bash
   chmod +x TradingManager.app/Contents/MacOS/TradingManager
   ```

2. 检查系统安全设置：
   - 系统偏好设置 → 安全性与隐私 → 允许从以下位置下载的应用
   - 确保"允许从App Store和被认可的开发者"被选中

3. 如果提示"无法验证开发者"：
   - 系统偏好设置 → 安全性与隐私 → 通用
   - 点击"仍要打开"按钮

#### 手动授权
如果系统阻止运行，请执行：
```bash
sudo xattr -rd com.apple.quarantine TradingManager.app
```
